Curso de Html

Essa pagina tem como objetivo explicar as principais funções do HTML com exemplos comentados.

Estrutura básica do HTML

A estrutura do HTML é dividida em algumas tags principais. A tag "head" contém metadados e configurações da página, como o título, codificação de caracteres, links de estilo (CSS) e outras informações que não são exibidas diretamente ao usuário. Já a tag "body" representa o corpo da página e inclui todo o conteúdo visível ao usuário, como títulos, parágrafos, imagens, tabelas e formulários, organizados por outras tags HTML.

Tags de texto

As tags de texto em HTML são usadas para estruturar, formatar e destacar conteúdos escritos em uma página da web. Elas definem títulos, parágrafos, negrito, itálico, citações, código, entre outros.

Links e imagens

As tags de links e imagens são fundamentais para a navegação e a exibição visual de uma página web. A tag <a> é usada para criar hiperlinks, permitindo que o usuário navegue entre páginas ou seções, com o atributo href definindo o destino. Já a tag <img> é usada para inserir imagens e exige os atributos src (caminho da imagem) e alt (texto alternativo, importante para acessibilidade). Ambas contribuem para a interatividade e apresentação do conteúdo.

Listas

O HTML oferece diferentes tipos de listas para organizar conteúdos de forma estruturada. A tag <ul> representa uma lista não ordenada, com marcadores (bolinhas), enquanto <ol> representa uma lista ordenada, com números. Dentro dessas listas, cada item é definido pela tag <li>. Existe ainda a <dl> para listas de definição, usadas para pares de termos e descrições com <dt> e <dd>. As listas são úteis para menus, passos, tópicos e muito mais.

Tabelas

As tabelas são criadas com a tag <table> e servem para exibir dados organizados em linhas e colunas. Cada linha é representada por <tr>, os cabeçalhos por <th>, e os dados comuns por <td>. Para uma estrutura mais clara, usa-se também <thead>, <tbory> e <tfoot>. A construção de tabelas é útil para exibir relatórios, comparações e registros organizados visualmente.

Divisões e agrupamentos

As tags <div> e <span> são usadas para dividir e agrupar conteúdo de forma genérica. A <div> cria um bloco de conteúdo, ocupando toda a largura disponível, enquanto <span> é usada para agrupar elementos em linha. Apesar de não terem significado semântico, essas tags são amplamente usadas para aplicar estilos, estruturar layouts e manipular seções com CSS e JavaScript.

Formulários

Formulários permitem coletar dados do usuário e enviá-los para processamento. São criados com a tag <form>, e contêm elementos como <input>, <textarea>, <select>, <label> e <button>. Cada campo possui atributos específicos, como type, name, placeholder, entre outros. Eles são essenciais em cadastros, pesquisas, comentários e qualquer interação com o usuário.

Estilo e scripts

O HTML permite incorporar estilos e comportamentos por meio das tags <stylet> e <scriptt>. A tag <stylet> é usada no <headt> para definir regras CSS que controlam aparência (cores, fontes, espaçamento etc.). A tag <script> é usada para incluir código JavaScript, que adiciona interatividade e lógica à página (eventos, animações, validações etc.). Esses elementos são fundamentais para criar sites modernos e dinâmicos.

Semânticas HTML5L

As tags semânticas descrevem o significado do conteúdo que envolvem, facilitando a compreensão por navegadores, mecanismos de busca e leitores de tela. Exemplos incluem <header>, <main>, <section>, <article>, <aside> e <footer>. Elas substituem o uso excessivo de <div> e ajudam a criar páginas mais acessíveis, organizadas e com melhor desempenho em SEO (otimização para motores de busca).
